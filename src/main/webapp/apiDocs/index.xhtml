Multi-vew Symbol Dictionary API

Multi-view Symbol Dictionary API ドキュメントへようこそ。

Multi-View Symbolとは、会津大学”活性知識学講座”が開発する「物事について多視点的な説明をする、多種多様なユーザが理解可能な記号の集合」です。

Multi-view Symbol Dictionary APIはMulti-view Symbolを提供するRESTfulなインターフェイスです。APIはJSON形式にエンコードされたオブジェクトを返します。

Search Images

GEThttps://{host_name}/api/multiview_symbol/{caption}/

Parameters

caption (必須) str 取得するMulti-view Symbolのセットを指定する
size str View Symbolの画像サイズ
Accepted values "small", "middle", "large"
Default middle
num int 取得するView Symbolの件数。最大件数以上の数を指定した場合最大件数取得。
Accepted values 1~最大件数
Default 最大件数
object str Clarifier “Object” が付加されたView Symbolを取得対象とする
Accepted values "all", "natural", "artificial"
time str Clarifier “Time” が付加されたView Symbolを取得対象とする
Accepted values "all", "time_of_day", "day", "week", "month", "year", "season"
place str Clarifier “Place” が付加されたView Symbolを取得対象とする
Accepted values "all", "position", "space"
situation int Clarifier “Situation” が付加されたView Symbolを取得対象とする
Accepted values "all", "activity", "status", "scene"
individual int Clarifier “Individual” が付加されたView Symbolを取得対象とする
Accepted values "all", "one", "single"
collective str Clarifier “Collective” が付加されたView Symbolを取得対象とする
Accepted values "all", "pair", "many"

Example

Captionが”beautiful”のMulti-view Symbolセットで、Clarifierに最低でもObject(all)とSituation(statusのみ)を含むView Symbolを最大で２０件取得する

https://{host名}/api/multiview_symbol?caption=beautiful&num=20&object=all&situation=status

Response keyDescription

clarifierTypes検索対象のClarifier
total_viewSymbolsMulti-view Symbolが保持するView Symbolの数

sizeView Symbolの画像サイズ
viewSymbolsView Symbolのリスト
idView Symbolのunique identifier
nameView Symbolの名前
imageURLView Symbol画像URL
clarifierTypesView Symbolに注釈されているClarifier
